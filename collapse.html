Collapse

El complemento JavaScript de colapso se utiliza para mostrar y ocultar contenido. Los botones o anclas se utilizan como disparadores que se asignan a elementos específicos que alterna. Al contraer un elemento, se animará la altura desde su valor actual hasta 0.

Collapse

Un --button-- o --a-- puede mostrar y ocultar varios elementos al hacer referencia a ellos con un selector en su atributo href o data-bs-target. Múltiples --button-- o --a-- pueden mostrar y ocultar un elemento si cada uno hace referencia a él con su atributo href o data-bs-target
